# Product Purchase API – OpenAPI Specification

This repository contains the **OpenAPI specification** for a REST API that calculates product prices and processes purchases.

The specification defines the API contract, including endpoints, request and response schemas, validation rules, and error handling. It can be used for backend implementation, client integration, and API documentation generation.

Interactive API documentation is available via **Swagger UI**.

## API Overview

The API provides two main operations:

- **Price Calculation** – calculates the final product price including discounts and taxes.
- **Purchase Processing** – performs a purchase using a payment processor.

All endpoints use **JSON request/response format** and follow standard HTTP status codes.
